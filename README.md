## Professional Summary

<p align="justify">
Saya adalah seorang developer yang bergerak di persimpangan antara <strong>Web Development</strong>, <strong>Data Science</strong>, dan <strong>Digital Infrastructure</strong>. Fokus utama saya adalah merancang aplikasi web yang scalable, melakukan pemrosesan data analitis yang mendalam, serta mengelola infrastruktur IT secara efisien dan optimal.
</p>

<p align="justify">

</p>
